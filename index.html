Welcome to GitHub Pages.

This project concern the creation of a Wifi remote control car, with Android application. Rather than other project available on the Web, the project do not use Linux, and you have full control on direction and speed, not just binary choice (Full speed or stop, full right, full left, ...). Skills used :

  • Real-time development, with OS (FreeRTOS)
  • Multi-task
  • Network communication (UDP,TCP)
  • Control algorithm
  • Hardware development (sensor)
